About Us

We are a senior capstone team from Wayne State University, our goal with this project was to create a way to scrape and aggregate unstructured data from online news sources, and translate that data through NLP techniques into a structured and legible format on the front end.

Functionality

Currently the web application allows users to search for events either on a map or table. The map makes the ability to find events by area both easy and interactive while the table grants the user ample information in an easy to read way.

Technologies Used

  • Selenium
